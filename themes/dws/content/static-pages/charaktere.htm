Pirat Dummwiestroh

Der kleine Pirat Dummwiestroh ist der Held unserer Geschichten. Dummwiestroh ist der Sohn des gefürchteten Piraten Königs Rotbart. Dummwiestroh ist nicht gerade der geschickteste und ist ziemlich tollpatschig. Aber egal was ihm passiert er gibt nicht auf und macht weiter.

{% endput %} {% put Rotbart %}

Rotbart

Rotbart ist der König der Piraten und der Vater von Dummwiestroh. Er hasst weiße Herzchen. auch wenn er gefürchtet ist auf der ganzen Welt ist er ein strenger aber liebender Vater der alles für seinen Sohn machen würde.

{% endput %} {% put Semmelknoedel %}

Semmelknödel

Semmelknödel ist Koch von Beruf. Er erfindet immer außergewöhnliche Gerichte und kocht für die ganze Crew. Er ist ein wenig dicker als die anderen und möchte am liebsten wie ein Französischer Sternekoch aussehen.

{% endput %} {% put Krummsaebel %}

Krummsäbel

Krummsäbel ist ein rauer Zeitgenosse. Er ist zuständig für die Kanonen und ärgert am liebsten den Koch Semmelknödel was immer wieder zu Problemen führt.

{% endput %} {% put Rumbert %}

Rumbert der Bär

Wie sein Name schon sagt ist er ein Bär von einem Pirat. Aber so groß und stark er auch ist, so ruhig und gemütlich ist er auch. Weil er so groß ist muss er immer was essen weil sonst sein Magen so laut knurrt wie ein Bär.

{% endput %} {% put Harry %}

Der verrückte Harry

Ein wenig verrückt und verwirrt aber immer lustig und freundlich. Ein Nudelsieb auf seinem Kopf ist sein Piratenhut den er nie absetzen würde.

{% endput %} {% put Adreana %}

Adreana

Adreana hat 5 große Brüder und ist ein wildes und abenteuerliches Mädchen. Sie ist die Stellvertreterin von Dummwiestroh wenn er nicht auf seinem Schiff ist und mag es gar nicht wenn sie unterschätzt wird.

{% endput %} {% put Helene %}

Helene Gnädig

Die Mutter von Dummwiestroh und die Frau von Rotbart. Helene Gnädig ist die gute Seele des Piratenkönigreiches. Sie schupft alleine das Königreich wenn Rotbart auf Raubzügen ist. Sie hilft überall wo sie nur kann und eigentlich weiß jeder dass sie die Chefin ist.

{% endput %} {% put Emil %}

Emil Adlerauge

Ist wie Dummwiestroh ein wenig ungeschickt, aber nur weil er ein wenig schlecht sieht was nah ist jedoch sieht er wie ein Adler was so in der weite passiert. Er hat eine super coole Jacke wo ein Adler abgebildet ist.

{% endput %} {% put Gerlinde %}

Gerlinde

Gerlinde ist der Hund von Rotbart und liebt es abzuhauen. Gerlinde ist ein freundlicher Hund der immer gute Laune verbreitet, liebt es zu kuscheln und Leckerlis.

{% endput %} {% put Ironimus %}

Ironimus Lenkrad

Ist der Steuermann. Er ist zwar schon älter aber keiner kennt die Weltmeere so gut wie er. Er kennt alle Legenden und Sagen und kann sehr oft gute Tipps geben wenn es heikel wird.

{% endput %} {% put Willi %}

Willi Faulsack

Wie sein Name schon verrät ist er nicht der schnellste und motivierteste. Am besten kann Willi Faulsack sich entspannen. Da macht ihm keiner was vor.

{% endput %} {% put Hannibal %}

Hannibal Locke

Ein großer Pirat mit roten Locken. Immer gut drauf und liebt seine Locken.

Antonio Fidelio

Ein italienischer Pirat. Er liebt es sich seine Haare zu machen und seine Bauchmuskeln zu trainieren. Wenn er redet muss er seine Hände dabei benutzen.
